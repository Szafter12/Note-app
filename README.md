## Run Locally

Clone the project

```bash
  git clone https://github.com/Szafter12/Note-app.git
```

Go to the project directory

```bash
  cd Note-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
